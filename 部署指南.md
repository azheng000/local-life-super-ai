## 方案一：Railway 部署（推荐）

### 步骤
1. 浏览器打开 https://railway.app
2. 点击 **Login with GitHub** 登录
3. 点击 **New Project** → **Deploy from GitHub repo**
4. 选择 `local-life-super-ai`
5. 在 **Variables** 添加：
   ```
   DATABASE_URL = postgresql://neondb_owner:npg_bsgDcCLJ54Mu@ep-hidden-forest-ae532vl5-pooler.c-2.us-east-2.aws.neon.tech/neondb?channel_binding=require&sslmode=require
   PORT = 3000
   NODE_ENV = production
   ```
6. 点击 **Deploy**
7. 等待 2-3 分钟构建完成
8. 复制生成的 URL（如 https://xxx.railway.app）

## 方案二：Render 部署（免费）

### 步骤
1. 浏览器打开 https://render.com
2. 点击 **Sign Up** 用 GitHub 登录
3. 点击 **New** → **Web Service**
4. 选择 `local-life-super-ai` 仓库
5. 配置：
   - Region: Singapore
   - Plan: Free
   - Build Command: `npm install && npm run build`
   - Start Command: `npm start`
6. 添加环境变量（同上）
7. 点击 **Deploy**

## 前端部署（Cloudflare Pages）

后端部署完成后，执行：
```bash
cd frontend
npm install
npm run build
```

然后将 `dist` 目录部署到 Cloudflare Pages。
